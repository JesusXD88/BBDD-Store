Avant de vous lancer :

Si vous avez le moindre doute sur la compatibilité de votre console, ou si vous possédez un modèle CECH-25xxx avec un date code 1B, utilisez minVerChk afin de vous assurer des possibilités. La version indiquée par minVerChk doit être 3.56 maximum. Toute tentative sur une console non supportée entrainera un brick. Ne pas utiliser PS3Xploit sur une console en CFW, ce sera également un brick assuré. Les "FAT" modèles CECHA0x, CECHB0x, CECHC0x, CECHE0x et CECHG0x ont des mémoires flash de type NAND. Toutes les autres "FAT" ou "Slim" compatibles au jailbreak ont des mémoires flash de type NOR. Votre console doit impérativement avoir son OFW en 4.82. Toute autre version ne fonctionnera pas et peut entrainer un brick (même si il y a des sécurités pour éviter ça). Mettez à jour votre console le cas échéant.
